# AI-Agent-Learning

Learning AI Agents using Ollama, LangChain, ChromaDB and Open Source LLMs.

## Technologies Used

- Python
- Ollama
- LangChain
- ChromaDB
- HuggingFace Embeddings
- RAG (Retrieval Augmented Generation)
- Vector Databases
